# Recording Pack

Single-place staging folder for the 3-minute competition recording.

## Structure

- `01-build-all/` - scene-1 command and terminal markers
- `02-waveforms/` - coupled-neuron waveform assets (SVG + PNG)
- `03-innovus/` - DEF/GDS/report artifacts for implementation scene
- `04-slides/` - title and close text cards
- `RECORDING-CHECKLIST.md` - pre-flight + run-order checklist

## Refresh Command

```bash
scripts/build_recording_pack.sh
```
