# Recording Checklist

## Pre-Flight

- [ ] Close unrelated windows and notifications.
- [ ] Terminal theme is high-contrast and readable.
- [ ] Terminal font is large enough for video (>= 18pt equivalent).
- [ ] Terminal window is full-screen at 16:9 aspect.
- [ ] Microphone level is tested (no clipping, no fan noise).
- [ ] `competition/voiceover-script.md` is open.
- [ ] `competition/video-shot-script.md` is open.

## Asset Check

- [ ] `competition/recording-pack/02-waveforms/neuro_tile4_coupled_spikes.svg`
- [ ] `competition/recording-pack/02-waveforms/neuro_tile4_coupled_mems.svg`
- [ ] `competition/recording-pack/03-innovus/alu4_flow_demo.gds`
- [ ] `competition/recording-pack/03-innovus/alu4_flow_demo.def`

## Dry Run

- [ ] `DEMO_SKIP_LONG=1 scripts/demo_narrator.sh` completes cleanly.
- [ ] Scene transitions feel natural with your speaking pace.

## Final Recording

- [ ] Run `scripts/demo_narrator.sh` for guided one-take capture.
- [ ] Keep cursor movement minimal and intentional.
- [ ] Pause 1-2 seconds at each scene banner before speaking.
