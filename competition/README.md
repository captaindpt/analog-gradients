# IC Taiwan Grand Challenge (ICTGC)

> Taiwan government competition to attract global IC startups to Taiwan's semiconductor ecosystem

---

## Quick Summary

| | |
|---|---|
| **Prize** | USD $30,000 + semiconductor resources |
| **Deadline** | February 28, 2026 (Round 4) |
| **Website** | [ictaiwanchallenge.org](https://ictaiwanchallenge.org) |
| **Award Ceremony** | COMPUTEX/InnoVEX, June 2-5, 2026 |

---

## Why This Matters for Our Project

Your **neuromorphic GPU core** project could fit under:
- **AI Core Technologies and Chips** - Hardware acceleration, AI chip design
- **Smart Manufacturing** - IC process innovation

Taiwan offers access to:
- **TSMC, UMC, PSMC** shuttle services
- **Cadence, Synopsys, Keysight** EDA tools
- **ARM, Andes** IP licensing
- Industry mentorship and VC connections

## Application Requirements

1. **Concept Paper** - Core technology, challenges solved, business model, market plan
2. **3-minute Pitch Video**
3. **IP Statement & Affidavit** - Signed declaration
4. **Online Form** - Organization info, product info, supplementary docs

---

## Judging Weights

- **40%** Local Connections (plans for Taiwan, industry development)
- **40%** Value Creation (innovation potential, economic value)
- **20%** Technological Innovation (emerging applications, cross-domain)
