# My Workspace

Reference documentation, planning, and session tracking.

Primary navigation is `my-workspace/docs/INDEX.md`.

## Onboarding Trail

For future agents/humans, use this order:

1. `AGENTS.md` (repo rules)
2. `my-workspace/README.md` (this contract)
3. `my-workspace/docs/INDEX.md` (single canonical trail source)
4. `my-workspace/docs/vision.md` (what we are building)
5. `my-workspace/docs/DEVELOPMENT.md` (how to execute)
6. `my-workspace/docs/STATUS.md` (where we are now)
7. `my-workspace/docs/RANDEZVOUS.md` (current operating loop)
8. `my-workspace/docs/armory/snapshot-2026-02-02/armory-summary.md` (tool reality)
9. `my-workspace/docs/reference/README.md` (secondary context)
