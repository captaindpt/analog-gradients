# My Workspace

Reference documentation, planning, and session tracking.

Primary navigation is `my-workspace/docs/INDEX.md`.

## Structure

```
my-workspace/
├── docs/
│   ├── INDEX.md          # Navigation + source-of-truth map
│   ├── vision.md         # Active product vision
│   ├── DEVELOPMENT.md    # Development workflow for agents
│   ├── STATUS.md         # Current progress tracker
│   ├── mani-plan.md      # Strategic roadmap
│   ├── gpu-spec.md       # GPU demo spec (minimal path)
│   ├── armory/           # Environment capability snapshots
│   └── reference/        # Technical reference docs
│       └── Cadence/      # Cadence setup guides
├── tickets/              # Work items and issues
└── logs/                 # Session logs
```

Format: `tickets/NNNN-short-description.md`

## Session Logs
